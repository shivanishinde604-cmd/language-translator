# 🌍 Language Translator

## 📌 Project Description
This is a simple web-based Language Translator application.  
It allows users to translate text between different languages using an online translation API.

The project is built using basic web technologies and helps users convert text quickly and easily.

---

## 🚀 Features
- Translate text between multiple languages
- Swap source and target languages
- Simple and user-friendly interface
- Instant translation using API

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- MyMemory Translation API

---

## 🌐 Supported Languages
- English
- Hindi
- Telugu
- Malayalam
- Kannada
- Tamil
- French
- Spanish
- Korean

---

## 📌 How It Works
1. Enter text in the input box  
2. Select source and target languages  
3. Click on Translate button  
4. Get the translated text instantly
